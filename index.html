⚙️ GitHub Setup

Your tasks will be saved as TXT files in your GitHub repository. Each month gets its own file: tasks_YYYY-MM.txt

  1. Create a new GitHub repository (can be private)
  2. Go to Settings → Developer → Personal Access Tokens
  3. Generate a Classic token with repo scope checked
  4. Paste the token and your repo details below
The repo must already exist on GitHub
Usually main or master
Stored only in your browser's localStorage — never sent anywhere except GitHub
Leave blank to store in repo root, or e.g. tasks for tasks/tasks_YYYY-MM.txt
